Clone the repo
Install all php dependencies - 'composer install'
Install all js dependencies - 'npm install'
Go to .env file and configure the application
	- App mode: 'online' or 'offline'
	- Admin email
	- email sender configuration
	- database details
	- database name 'ecare'
Run db migration - 'php artisan migrate'
Seed admin account, run - 'php artisan db:seed'
Start vite - 'npm run dev'
Start laravel - 'php artisan serve'

Enjoy!!!!
